### npm 
Node Package Manager (Node.js 기본 패키지 관리자)

### npm i, npm install 
설치명령어

### node_modules
라이브러리 모음폴더

### src폴더 안 app.js
메인페이지

### public 폴더안 index.html
메인페이지(app.js있는거 index.js로 index.html에 뿌려줌)

### package.json
설치한 라이브러리 목록


### <설치류>
npm i -g react-scripts

### <버전확인>
npm -v 
npx -v
node -v

### <react 프로젝트 생성>
npx create-react-app 플젝이름(대문자, 특수문자 X)

### <빌드>
npm start
npm run start
